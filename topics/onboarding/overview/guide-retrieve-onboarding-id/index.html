Retrieve an onboarding ID

There are two methods you can use to retrieve an onboarding ID.

  1. Run an API query to get the ID (you can also use this query to retrieve other information).
  2. Locate onboarding IDs on your Dashboard.
Prerequisites

To retrieve an onboarding ID, you must have an existing onboarding, either individual or company.
The onboarding can have any status.

API guide​

Queries are highly customizable. This guide focuses solely on using the onboardings query to retrieve onboarding IDs. Learn more about this query in the monitor onboarding progress guide.

  1. In the API Explorer, add the onboardings query.
  2. Check the boxes for edges and node.
  3. Under node, check the boxes for email and id. Retrieving the user's email with the id lets you know the user attached to the onboarding ID.
  4. Click ▶ Run.

Query​

🔎 Open the query in API Explorer

query RetrieveOnboardingId {
  onboardings {
    edges {
      node {
        email
        id
      }
    }
  }
}

Payload​

{
  "data": {
    "onboardings": {
      "edges": [
        {
          "node": {
            "email": "alberto.moreno@mimarca.io",
            "id": "eda0ceec-0e20-4d1b-bbee-b3e3a4227c99"
          }
        },
        {
          "node": {
            "email": "sofia.ramos@mimarca.io",
            "id": "ab9a5b1a-c24a-4f7c-ab7c-330a0a41ceea"
          }
        }
      ]
    }
  }
}

Dashboard​

All onboarding IDs for your project are on your Dashboard.

Go to Dashboard > Data > Onboardings. The only ID included on this list is the onboardingId.
