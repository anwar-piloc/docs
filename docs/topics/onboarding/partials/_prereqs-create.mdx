<p>There are two methods you can use to create {props.onboardingType} onboarding links.</p>
<ol>
<li><b>Run an API mutation</b> to create a unique onboarding link for each future {props.onboardingType} account holder.</li>
<li>Use a <b>public URL from your Dashboard</b> to share the same {props.onboardingType} onboarding link with all of your users.</li>
</ol>
</div>

:::tip Prerequisites
To create an onboarding link, you must be an account member with `canManageMemberships` permissions.
:::
